**Facts in short:**
- Compatible with version 5.7 and 8+
- One click installation
- Easy manage glossary items in your sitemap
- Can be used for generic A-Z organised navigations, not just for glossaries.
- Bonus: Contains custom templates for auto nav + page list blocks
- Works on all modern browsers and devices
- Fully Responsive
- Multilingual

**Watch the [video](https://www.youtube.com/watch?v=ELeaYvcIacs) to see the add-on in action!**

With the concrete5 add-on "Glossary List" it is very easy to explain unclear terminology in form of a glossary. Simply create an "Agenda" parent page in your site tree and add pages below for each term you want to explain. The sorting of the subpages does not matter and is not considered.

Once you have entered all the terminology, you can easily add the "Glossary List" block element to your page and select the parent page in the block settings. Once saved, a list grouped by the first letters, including a letter navigation above the list, will be displayed at the point where you placed the block element. If you click on a letter in the letter navigation, it automatically scrolls to the respective position of the list.

This extension is very useful for users who want to integrate a glossary into their site without much effort. Of course, the block element is responsive and runs on all modern browsers and devices. The add-on is fully translatable and available in German and English.
